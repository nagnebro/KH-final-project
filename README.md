## 프로젝트 소개

- 개발 환경 <br>
  프레임워크 : spring boot 2.7.6 <br>
  빌드 툴 : gradle <br>
  템플릿 엔진 : thymeleaf <br>
  JDK : Java 11 <br>
  DataBase : MySQL 8.0.33 (spring data JPA) <br>
  형상관리 : gitHub <br>
  이외 : HTML, CSS, JavaScript, Ajax, jquery <br>

- 개발 기간 : 2023.03 ~ 2023.09
  
  

## 주요기능

### ⭐️ 동행 찾기 및 채팅 기능
<ul>
  <li>여행을 가고 싶어하는 사람들이 서로 동행게시글을 작성함으로서 동행을 찾을 수 있습니다.</li>  
  <li>같이 여행가고 싶어하는 사용자에게는 채팅을 걸어 개인적으로 대화를 나눌 수 있습니다.(관리자 1:1 채팅도 가능)</li>
</ul>

## 화면구성

홈페이지의 카테고리는 크게 커뮤니티와 여행상점으로 나뉘어져있습니다.
커뮤니티안에 세부카테고리로는 일정과 동행으로 나뉘어지고 여행상점안에서는 여행지 둘러보기, 항공권, 렌트카로 나뉘어집니다.
